# 🧠 Neuro Train

Платформа когнитивных тренировок — 5 игр для развития памяти, внимания, реакции и логики.

## Стек

| Слой       | Технология                         |
|------------|-------------------------------------|
| Frontend   | Vanilla HTML / CSS / JavaScript     |
| Шрифты     | Orbitron + Outfit (Google Fonts)    |
| Сервер     | Python 3 + Flask                    |
| Деплой     | Gunicorn / любой хостинг статики    |
| Хранилище  | localStorage (на стороне клиента)   |

---

## Структура проекта

```
neuro-train/
├── index.html              # Главная страница (вся разметка)
├── server.py               # Flask-сервер для локальной разработки
├── requirements.txt
├── .gitignore
│
├── css/
│   └── style.css           # Все стили
│
├── js/
│   ├── data.js             # Константы: USERS, ACHS, SEQS, WORDS, COLS

│   ├── leaderboard.js      # Рендер таблицы лидеров
│   ├── profile.js          # Рендер страницы профиля
│   ├── utils.js            # nav(), toast(), shuffle(), timer, init
│   │
│   └── games/
│       ├── memory.js       # Memory Game (найди пары)
│       ├── color.js        # Color Reaction (реакция на цвет)
│       ├── word.js         # Word Association (ассоциации)
│       ├── numeric.js      # Numeric Sequence (числовые последовательности)
│       └── puzzle.js       # Puzzle Solver (пятнашки)
│
```

## Быстрый старт

### Вариант 1 — открыть напрямую в браузере

Просто дважды кликни на `index.html`. Работает без сервера.

> ⚠️ Из-за политики CORS некоторые браузеры (Chrome) могут блокировать загрузку локальных JS-файлов.
> В этом случае используй вариант 2.

### Вариант 2 — Flask (рекомендуется для разработки)

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск
python server.py
```

Открой в браузере: **http://localhost:5000**

### Вариант 3 — встроенный Python HTTP-сервер (без Flask)

```bash
python -m http.server 5000
```

Открой: **http://localhost:5000**

## Игры

| Игра                  | Навык          | Описание                                      |
|-----------------------|----------------|-----------------------------------------------|
| 🧠 Memory Game        | Память         | Найди все парные карточки                     |
| ⚡ Color Reaction      | Реакция        | Нажимай, когда цвет совпадает с целевым       |
| 💬 Word Association   | Ассоциации     | Пиши первое слово-ассоциацию к предложенному  |
| 🔢 Numeric Sequence   | Логика         | Найди следующее число в последовательности    |
| 🧩 Puzzle Solver      | Пространство   | Пятнашки — расставь плитки по порядку         |

---

## Безопасность

- Пароли **не хранятся в открытом виде** — используется SHA-256 (Web Crypto API) с солью
- Капча при регистрации (математическая задача) защищает от ботов
- Все данные хранятся локально в `localStorage` браузера

## Деплой на сервере (Gunicorn + Nginx)

```bash
# Установка
pip install -r requirements.txt

# Запуск через Gunicorn
gunicorn server:app --bind 0.0.0.0:8000 --workers 2

# Nginx: проксировать :80 → :8000
```
